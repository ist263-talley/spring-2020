Update your 'letter.html' page

Below are instructions

  • Open VS Code in the class' folder.
  • Copy the letter.html file from week-02 folder into the current lab folder
  • Create a css file named `style.css` in the labs folder
  • Link to the css file from your letter.html
  • Edit style.css to add the following CSS declaraions
    • Find a Google font you like and use it
    • Add bottom borders to different 'sections'
    • Add bottom margins to 'sections'
    • Set the background image for the page. You can use the image in 'images/gradient_bg.png'
  • Don't forget to reference the sample files Background, Fonts to see how its been done
