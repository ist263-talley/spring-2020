Group Activty / Lab

  • Download the file week-05/02/labs/week-05-02-lab.zip and expand it in a new subfolder on your PC.
  • Create a style.css file in the same labs folder and add a <link> element to resume.html to reference your CSS file.
  • Follow guidance in comments within resume.html to style the CSS so the page matches the file resume-template.png in the same folder.

The goal here is to create CSS that matches a design given to you by another source rather than choosing your own styling.
