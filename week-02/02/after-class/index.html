Letter Markup - Worth 5 points

The text for the letter is in letter.txt. You can copy/paste it into the HTML document

Deliverables

  • File name: [your SU NetID]-letter.html
  • Must include the tags listed below
  • Run the file through W3C Validator
  • Submit via Blackboard
You will need to keep this link handy for researching the tags - https://www.w3schools.com/tags/tag_comment.asp

Along with required HTML tags (remember which those are?) the following tags must be used

  • main, header, section, footer
  • time
  • abbr
  • dl, dt, dd
  • q
  • cite
  • sub, sup

Here is some additional information to consider

The letter in general should be marked up with a structure of paragraphs and headings, with some exceptions:

  • There is one top level heading (the "Re:" line) and three second level headings.
  • The semester start dates, study subjects and exotic dances should be marked up using an appropriate list type.
  • In addition, each line of the addresses should sit on a new line, but not break to a new paragraph.
